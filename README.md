# [pet-project] KyivRadaVotes

## Project Setup

1. Run virtual server for data emulation from `@\data` folder:

```sh
json-server --watch src/data/pytannia.json -p 2222
```

2. Install or update modules:

```sh
npm install
```

3. Compile and Hot-Reload for Development

```sh
npm run dev
```
